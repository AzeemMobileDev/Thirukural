## Introduction

**Thirukural** is a classic Tamil language text consisting of 1,330 short couplets, or kurals, of seven words each. This application contains the kurals and its explanations.

## Tech Stack

- **Language**: Kotlin
- **UI Toolkit**: Jetpack Compose
